- Node https://nodejs.org

## npm
Run server
```npm
npm start
```

Run server, compile webpack with -d flag, set watchers and add logger middleware
```npm
npm run dev
```

Compile webpack with -p flag and set NODE_ENV on 'production'
```npm
npm run production
```
